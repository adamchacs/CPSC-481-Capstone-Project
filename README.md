
# How to Run the A.I. Mystery Dungeon Project

This project was made in **Godot**, so you need to download Godot before running the game.

## 1. Download Godot

1. Go to the official Godot website:  
   https://godotengine.org/download

2. Download **Godot 4.x** for your operating system.

3. Install or open Godot.

For Windows, you can usually just unzip Godot and open the Godot application.  
For Mac, move Godot into your Applications folder.  
For Linux, unzip it and run the Godot executable.

## 2. Download the Project ZIP File

1. Go to the GitHub repository.
2. Click the green **Code** button.
3. Click **Download ZIP**.
4. Save the ZIP file to your computer.

## 3. Unzip the Project

1. Find the downloaded ZIP file.
2. Right-click the file.
3. Select **Extract All** or **Unzip**.
4. Open the extracted folder.

## 4.  Open the Project in Godot
1.Open Godot.
2.Click Import.
3.Click Browse.
4.Find the extracted project folder.
5.Select the file 

## 5. Run the Game
1.In Godot, open the scene:
Dungeon2.tscn
2. Click the Play button in the top-right corner. The game should start running.
